# Visualization Summary
This visualization shows the number of building permits approved by the City of Seattle for various types from 2004 to 2026. The data is from the Seattle Department of Construction and Inspections. The data from the city of Seattle show the dates the building permits were applied for and approved. I used the IssuedDate column from the City of Seattle dataset and the PermitClass column to construct my visualization. After pulling data from these two columns from the Seattle government website, I use Excel to add a new column labeled "Year". Then I imported this Excel file into a Google Colab Jupyter notebook to aggregate the number of permits issued by each type of building permit issued by the city of Seattle, and then created a new Excel file using Python for my visualization. The two Excel files mentioned are attached to this repository.
