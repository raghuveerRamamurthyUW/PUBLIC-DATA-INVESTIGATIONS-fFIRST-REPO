This story includes two visualizations of data collected by the Seattle Fire Department on the decommissioning of residential heating oil tanks. This story focuses specifically on the tank size data collected by the SFD. The first visualization shows inconsistencies in the data, including mismatched unit labels and missing values. The second visualization displays the standardized data, and the visualizations explain the dataset's flaws in depth. I cleaned the raw data from the source using Python several times to produce these visualizations.
